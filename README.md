## Project Overview
This project is a Java-based Vehicle Management System created to demonstrate core Object-Oriented Programming (OOP) principles such as inheritance, abstraction, composition, aggregation, and method overriding.

The system models different types of vehicles and their associated drivers using a clean and structured class hierarchy.

## OOP Concepts Used
- Inheritance: Car, Motorcycle, and Truck inherit from the abstract Vehicle class
- Abstraction: Vehicle is an abstract class with abstract and concrete methods
- Method Overriding: Each subclass provides its own implementation of engine methods
- Composition and Aggregation: Each Vehicle has a Driver, and one Driver can be associated with multiple Vehicles

## Class Hierarchy
- Vehicle (abstract superclass)
- Car
- Motorcycle
- Truck
- Driver
